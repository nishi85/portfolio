Brandit

Brandit is an interactive app allowing a user to auto-generate branding for their company. Using the Noun-Project API, the user inputs keywords associated with their company and in turn a logo is generated. The user is able to customize their logo with choice of colour and emblem, and save their choice to their profile using Firebase. Made in collaboration with Manon Rugge and Callie Scott.

React / Firebase / API / JavaScript / Sass
